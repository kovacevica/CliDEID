This distribution contains one program:

	* CliDEID a command line tool for the de-identification of clinical narratives. 

The software can be run from the root folder, and is explained, with small examples, in the guide file in the /doc folder.

CliDEID uses and comes bundled with three third party tools:
--Apache cTAKES http://ctakes.apache.org/ - used for natural language processing
--CRF++ https://code.google.com/p/crfpp/ - used for machine learning
Please see the appropriate folders and links for copyright notices of each of these tools.
